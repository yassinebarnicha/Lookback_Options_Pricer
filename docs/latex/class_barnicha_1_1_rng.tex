\doxysubsection{Detailed Description}
RNG wrapper providing N(0,1) and U(0,1). 

We use {\ttfamily std::\+mt19937} (Mersenne Twister) because it is:\+
\begin{DoxyItemize}
\item part of the C++ standard library,
\item reproducible across platforms (given the same seed),
\item fast, with a very long period.
\end{DoxyItemize}

Note:\+ {\ttfamily \doxylink{class_barnicha_1_1_rng_ad8a8cf8e56b9d725ff1776da4a90f1f2}{next\+\_\+uniform()}} is clipped away from 0 to avoid {\ttfamily log(0)} in inversion formulas.
