Exported DLL function to price a lookback option. 

This function serves as the main external interface of the lookback pricing DLL. It selects the appropriate pricing method based on the input parameter {\ttfamily method} and delegates computation to the corresponding implementation function.

Supported pricing methods:\+
\begin{DoxyItemize}
\item method = 0 → Monte Carlo pricing with optional BGK correction
\item method = 1 → Conditional expectation pricing method
\end{DoxyItemize}

The computed results (price and optionally standard errors or Greeks) are written into the provided output buffer.
