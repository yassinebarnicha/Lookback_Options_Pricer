\doxysubsection{Detailed Description}
Closed-\/form price for floating-\/strike lookback options (continuous monitoring). 

Includes a dedicated {\ttfamily rate == 0} limit to avoid numerical issues when the standard formula contains terms proportional to 1/\+r.
