This project provides a {\bfseries{unified pricing DLL}} for {\bfseries{European floating-\/strike lookback options}} under the {\bfseries{Black–\+Scholes}} model.

The library integrates {\bfseries{two independent Monte Carlo approaches}} developed separately and compiled into a {\bfseries{single shared library}}:\+


\begin{DoxyItemize}
\item {\bfseries{\doxylink{namespace_barnicha}{Barnicha} Engine}} ~\newline
 One-\/step Monte Carlo with exact extremum sampling ({\bfseries{continuous monitoring}}, {\bfseries{no discretization bias}})
\item {\bfseries{\doxylink{namespace_chivet}{Chivet} Engine}} ~\newline
 Time-\/stepping Monte Carlo with optional {\bfseries{BGK}} bias correction
\end{DoxyItemize}

Both engines are accessible through a {\bfseries{single unified DLL interface}} and can be used interchangeably for pricing and Greeks computation.

\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{Engine A — Barnicha (Exact One-\/\+Step Monte Carlo)}{Engine A — Barnicha (Exact One-\/\+Step Monte Carlo)}}\label{index_autotoc_md5}
\hypertarget{index_autotoc_md6}{}\doxysubsection{\texorpdfstring{Core idea (no discretization bias)}{Core idea (no discretization bias)}}\label{index_autotoc_md6}
Let \$\$ ( X\+\_\+t = \textbackslash{}log S\+\_\+t ) \$\$. Under Black–\+Scholes:\+

\$\$ X\+\_\+t = \textbackslash{}log S\+\_\+0 + \textbackslash{}left(r-\/\textbackslash{}frac\{1\}\{2\}\textbackslash{}sigma\texorpdfstring{$^\wedge$}{\string^}2\textbackslash{}right)t + \textbackslash{}sigma W\+\_\+t. \$\$

We simulate the terminal value exactly:\+

\$\$ X\+\_\+T = \textbackslash{}log S\+\_\+0 + \textbackslash{}left(r-\/\textbackslash{}frac\{1\}\{2\}\textbackslash{}sigma\texorpdfstring{$^\wedge$}{\string^}2\textbackslash{}right)T + \textbackslash{}sigma \textbackslash{}sqrt\{T\}\textbackslash{},Z, \textbackslash{}qquad Z\textbackslash{}sim\textbackslash{}mathcal N(0,1). \$\$

Conditional on \$\$ ( (X\+\_\+0, X\+\_\+T) ) \$\$ , the process is a Brownian bridge; the conditional distribution of its running maximum is:\+

\$\$ \textbackslash{}mathbb P\textbackslash{}left(\textbackslash{}max\+\_\+\{0\textbackslash{}le t\textbackslash{}le T\}X\+\_\+t \textbackslash{}le m \textbackslash{}mid X\+\_\+0=x\+\_\+0, X\+\_\+T=x\+\_\+T\textbackslash{}right) = 1-\/\textbackslash{}exp\textbackslash{}left(-\/\textbackslash{}frac\{2(m-\/x\+\_\+0)(m-\/x\+\_\+T)\}\{\textbackslash{}sigma\texorpdfstring{$^\wedge$}{\string^}2T\}\textbackslash{}right),\textbackslash{}quad m\textbackslash{}ge\textbackslash{}max(x\+\_\+0,x\+\_\+T). \$\$

Inversion gives a quadratic equation; we take the root \$\$( m\textbackslash{}ge \textbackslash{}max(x\+\_\+0,x\+\_\+T) )\$\$. ~\newline
 The minimum is obtained by symmetry (the other root \$\$ ( \textbackslash{}le \textbackslash{}min(x\+\_\+0,x\+\_\+T) )\$\$).

Antithetic variates are applied {\bfseries{only}} to the normal draw used for the terminal value:\+

For each sample \$\$( (Z, U\+\_\+\{\textbackslash{}max\}, U\+\_\+\{\textbackslash{}min\}) )\$\$, we also evaluate ~\newline
 \$\$( (-\/Z, U\+\_\+\{\textbackslash{}max\}, U\+\_\+\{\textbackslash{}min\}) ) \$\$and average the payoffs.

\$\$ 0=t\+\_\+0 \texorpdfstring{$<$}{<} t\+\_\+1 \texorpdfstring{$<$}{<} \textbackslash{}dots \texorpdfstring{$<$}{<} t\+\_\+N = T, \textbackslash{}qquad \textbackslash{}\+Delta t = \textbackslash{}frac\{T\}\{N\}. \$\$

\hypertarget{index_autotoc_md20}{}\doxysubsection{\texorpdfstring{BGK correction (optional, used for pricing)}{BGK correction (optional, used for pricing)}}\label{index_autotoc_md20}
Discrete monitoring misses extrema between grid points. ~\newline
 With BGK enabled, the observed extrema are adjusted before computing the payoff:\+

Minimum (BGK correction):\+

\$\$ S\+\_\+\{\textbackslash{}min\}\texorpdfstring{$^\wedge$}{\string^}\{\textbackslash{}mathrm\{corr\}\} = S\+\_\+\{\textbackslash{}min\}\texorpdfstring{$^\wedge$}{\string^}\{\textbackslash{}mathrm\{disc\}\} \textbackslash{}exp\textbackslash{}left(-\/\textbackslash{}beta \textbackslash{}sigma \textbackslash{}sqrt\{\textbackslash{}\+Delta t\}\textbackslash{}right) \$\$

\$\$ S\+\_\+\{\textbackslash{}max\}\texorpdfstring{$^\wedge$}{\string^}\{\textbackslash{}mathrm\{corr\}\} = S\+\_\+\{\textbackslash{}max\}\texorpdfstring{$^\wedge$}{\string^}\{\textbackslash{}mathrm\{disc\}\} \textbackslash{}exp\textbackslash{}left(\textbackslash{}beta \textbackslash{}sigma \textbackslash{}sqrt\{\textbackslash{}\+Delta t\}\textbackslash{}right) \$\$

where the constant is:\+

\$\$ \textbackslash{}beta \textbackslash{}approx 0.\+5826. \$\$

Then the payoff uses the corrected extrema:\+
