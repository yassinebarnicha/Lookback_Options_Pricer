\doxysubsection{Detailed Description}
Pricing engine interface (Strategy pattern). 

Each engine exposes the {\itshape samples} of its estimator, via discounted\+\_\+payoffs():\+
\begin{DoxyItemize}
\item Monte Carlo:\+ one discounted payoff per path
\item Analytic :\+ a single sample \{exact price\}
\end{DoxyItemize}

This lets Greeks and reporting compute standard errors uniformly, without branching on the concrete engine type.
