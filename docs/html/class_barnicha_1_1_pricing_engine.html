Detailed Description

Pricing engine interface (Strategy pattern).

Each engine exposes the samples of its estimator, via discounted_payoffs():

  • Monte Carlo: one discounted payoff per path
  • Analytic : a single sample {exact price}

This lets Greeks and reporting compute standard errors uniformly, without branching on the concrete engine type.
