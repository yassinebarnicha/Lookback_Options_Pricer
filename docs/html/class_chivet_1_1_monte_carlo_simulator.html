Detailed Description

Generates spot-price paths under Geometric Brownian Motion (GBM).

The underlying dynamics are:

\[ S_{t+dt} = S_t \exp\!\bigl[(r - \tfrac{1}{2}\sigma^2)\,dt + \sigma\sqrt{dt}\,Z\bigr], \quad Z \sim \mathcal{N}(0,1) \]

Each path tracks the spot at maturity and the running maximum / minimum, which are needed for lookback option payoffs.
